## GF(3) Assignment Rule
- `id mod 3 == 0` -> trit=0, color=#d3869b (pink), name=ERGODIC
- `id mod 3 == 1` -> trit=+1, color=#b8bb26 (yellow-green), name=PLUS
- `id mod 3 == 2` -> trit=-1, color=#cc241d (red), name=MINUS
